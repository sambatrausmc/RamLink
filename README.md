# RamLink

RamLink is a campus club discovery and management app for Farmingdale State College students, club officers, and administrators. The app helps students explore clubs, view events, save opportunities, manage notifications, and track their profile. It also gives club officers workspace pages for managing events, announcements, resources, members, join requests, and student inquiries. Admin pages support reviewing clubs, users, and reported content.
